# Technical Guide

## System Architecture
1. GUI Layer (zmtech_main.py)
2. Analysis Engine (analysis_engine.py)
3. Data Processing Layer
4. Visualization Engine

## Components
### GUI Layer
- Built with Tkinter
- Handles user input
- Displays results
- Manages report generation

### Analysis Engine
- Stock data retrieval
- Technical analysis
- Statistical calculations
- Results processing

### Data Processing
- Data cleaning
- Normalization
- Feature extraction
- Error handling

### Visualization
- Price charts
- Volume analysis
- Correlation plots
- Technical indicators
